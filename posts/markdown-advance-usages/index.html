0x03 LaTex 数学公式

可以通过使用 $ 包裹行内公式, 或者使用 $$ 包裹多行公式, 来实现渲染 Latex 数学公式, 下面是一些例子

  • $E=mc^2$ 被渲染为: $E=mc^2$
  • $\Gamma(n) = (n-1)!\quad\forall n\in\mathbb N$ 被渲染为: $\Gamma(n) = (n-1)!\quad\forall n\in\mathbb N$
$$
\Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,.
$$

渲染结果为:

$$ \Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,. $$

0x04 UML 时序图

比如这样写:

​```sequence
participant Andrew
participant China
Andrew-China: Says Hello
Note right of China: China thinks\nabout it
China--Andrew: How are you?
Andrew-China: I am good thanks!
​```

得到的时序图为:

participant Andrew
participant China
Andrew-China: Says Hello
Note right of China: China thinks\nabout it
China--Andrew: How are you?
Andrew-China: I am good thanks!

更多语法请参考官方文档: https://bramp.github.io/js-sequence-diagrams/

0x05 Flowchart 流程图

比如这样写

​```flow
st=>start: Start
e=>end
op=>operation: My Operation
cond=>condition: Yes or No?

st->op->cond
cond(yes)->e
cond(no)->op
​```

得到的流程图为:

st=>start: Start
e=>end
op=>operation: My Operation
cond=>condition: Yes or No?

st->op->cond
cond(yes)->e
cond(no)->op

更多语法请参考官方文档: http://flowchart.js.org/

0x06 Graphviz

比如这样写:

digraph G {
  main -> parse -> execute;
  main -> init;
  main -> cleanup;
  execute -> make_string;
  execute -> printf
  init -> make_string;
  main -> printf;
  execute -> compare;
}

渲染出来的图为:

digraph G {
  main -> parse -> execute;
  main -> init;
  main -> cleanup;
  execute -> make_string;
  execute -> printf
  init -> make_string;
  main -> printf;
  execute -> compare;
}

更多语法请参考官方文档: https://www.graphviz.org/
